### 3.运行

- 安装

```
$ git clone git@github.com:CookChopper/VueMusicPcApp.git
$ npm install
```
或者
```
$ git clone https://github.com/CookChopper/VueMusicPcApp.git
$ npm install
```

- 运行
```
$ npm run serve
```
